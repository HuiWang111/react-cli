# setup-react-env

## install
```shell
npm i setup-react-env -g
```

## usage
### to create react app
```shell
setup-react-env create
# or
sre create
```

### version
```shell
setup-react-env --version
# or
sre --version
```

### help
```shell
setup-react-env --help
# or
sre --help
```
